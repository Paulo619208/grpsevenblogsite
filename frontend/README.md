# Frontend (React + React Bootstrap)

## Setup
1. Install dependencies:
   - `npm install`
2. Copy `.env.example` to `.env` if needed.
3. Run:
   - `npm start`

## Notes
- Uses Bootswatch `Litera` theme + React Bootstrap.
- Fetches the published article from backend API endpoint:
  - `GET /api/articles/payment-methods`
